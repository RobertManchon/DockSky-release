# Démarrage rapide

## 1. Créer un projet

Depuis le tableau de bord, cliquez sur **Nouveau projet**.

Renseignez :
- **Nom** du projet
- **Description** (optionnel)
- **Date cible** (optionnel)

## 2. Ajouter des étapes

Un projet est divisé en **étapes** (phases de travail). Par exemple :
- Conception
- Développement
- Tests
- Livraison

## 3. Ajouter des actions

Dans chaque étape, ajoutez des **actions** — ce sont les tâches concrètes à réaliser.

<Tip>
  Une bonne action est courte, concrète et réalisable en une session de travail.
</Tip>

## 4. Lancer un focus

Sur n'importe quelle action dans la Roadmap, cliquez sur le bouton **▶ Focus** pour démarrer le chrono.

Vous pouvez enchaîner plusieurs sessions Pomodoro — le temps réel de travail (sans les pauses) est cumulé et enregistré automatiquement quand vous marquez l'action comme terminée.

## 5. Marquer comme terminé

Cliquez sur le cercle à gauche de l'action pour la passer en **DONE**. Le temps accumulé est sauvegardé.
