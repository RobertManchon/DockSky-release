# Tour de l'interface

DockSky est organisé autour de deux éléments permanents (le bandeau et le menu) et d'une zone de contenu principale qui change selon la vue active.

---

## Le Bandeau permanent

Le bandeau est **toujours visible**, quel que soit ce que tu fais. Il est ancré en haut de la fenêtre et reste accessible depuis n'importe quelle vue.

Il remplit deux rôles :
- **Capture rapide** : saisir une tâche, une idée ou un rappel sans changer de vue
- **Information permanente** : heure, notifications en attente, session Pomodoro active

→ Voir [Bandeau permanent](../fonctionnalites/bandeau-permanent) pour le détail complet.

---

## Le Menu de navigation

Le menu latéral donne accès à toutes les vues de DockSky. Il est organisé en **5 groupes** :

### Vue d'ensemble
| Vue | Rôle |
|-----|------|
| 🏠 **Dashboard** | Vue d'accueil — stats du jour, actions en retard |
| 🔔 **Notifications** | Alertes, retards, rappels arrivés à échéance |
| ⏰ **Rappels** | Gestion des rappels horodatés |
| 📅 **Agenda** | Vue mensuelle des événements |
| 🗺️ **Roadmap** | Progression de tes projets |
| ✅ **Liste des accomplis** | Historique de tes victoires |

### Productivité
| Vue | Rôle |
|-----|------|
| 🍅 **Pomodoro** | Timer de focus avec statistiques |
| 🔁 **Routines** | Habitudes structurées et Watchdog |
| ⚡ **Exécution** | Mode une-tâche-à-la-fois |

### Organisation
| Vue | Rôle |
|-----|------|
| 📁 **Projets** | Gestion de tes projets |
| ☑️ **Tâches** | Liste complète de tes tâches |
| 📋 **Backlog** | Vue filtrée de toutes les tâches |
| 🧠 **Facettes** | Mémoire structurée par projet |
| 🤖 **Contextes IA** | Composition de contextes pour ton IA |
| 🗺️ **Parcours** | Timeline chronologique de ta progression |
| 📓 **Journal** | Carnet de bord de session |

### Capture
| Vue | Rôle |
|-----|------|
| 💡 **Idées** | Carnet d'idées brutes |
| 🆓 **Actions libres** | Tâches capturées, pas encore planifiées |
| 📚 **Knowledge** | Base de connaissances personnelle |
| 📁 **Activité fichiers** | Fichiers récemment modifiés |

### Compte
| Vue | Rôle |
|-----|------|
| 👤 **Profil** | Informations de compte |
| 🏆 **Achievements** | Badges et gamification |
| ⚙️ **Paramètres** | Configuration de l'app |

---

## Les modes d'affichage

DockSky propose plusieurs modes de navigation, sélectionnables depuis les paramètres :

- **Classic** — accès complet à toutes les sections, menu latéral visible
- **Weather** — vue épurée avec focus du jour et indicateur météo projets
- **Zen** — vue minimaliste, distractions réduites au maximum

Dans tous les modes, le bouton **⋯** donne accès à toutes les vues disponibles.

---

## Raccourcis clavier globaux

| Raccourci | Action |
|-----------|--------|
| `Ctrl+Shift+B` | Signaler un bug sur la vue active |
| *(configurable)* | Activer le bandeau permanent depuis n'importe quelle app |

Les raccourcis globaux fonctionnent même quand DockSky est en arrière-plan.
