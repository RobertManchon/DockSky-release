# Tour de l'interface

:::note En cours de rédaction
Documentation en cours de rédaction — cette section sera enrichie prochainement.
:::

## Les trois modes

DockSky propose trois modes de navigation, sélectionnables à tout moment :

- **Classic** — vue complète, accès à toutes les sections
- **Weather** — vue épurée avec météo et focus du jour
- **Zen** — vue minimaliste, distractions réduites au maximum

## Le bandeau permanent

Le bandeau est toujours visible, quel que soit le mode actif. Il te permet de capturer un projet, une tâche ou une étape sans interruption.

## Le menu

Le menu est organisé en 5 groupes. Un bouton **⋯** donne accès à toutes les vues, identique dans les trois modes.
