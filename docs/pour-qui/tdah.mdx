# DockSky pour les cerveaux TDA

:::note En cours de rédaction
Cette section sera disponible avant la sortie de la version publique de DockSky.
:::

DockSky est né d'un cerveau TDA, pour répondre à des problèmes réels — pas théoriques. Chaque choix de design a été pensé pour réduire la friction, pas pour impressionner.
