# Pomodoro

Le chrono Pomodoro de DockSky vous permet de travailler en sessions de travail entrecoupées de pauses. Seul le temps de travail effectif est comptabilisé.

## Démarrer une session

Cliquez sur le widget Pomodoro dans le bandeau de l'application. Sélectionnez un preset (25/5 min par défaut) et lancez la session.

## Associer une session à une action

Avant de démarrer, vous pouvez associer la session à un projet et une action. Le temps sera alors automatiquement cumulé sur cette action.

## Enchaîner plusieurs Pomodoros

Vous pouvez enchaîner plusieurs sessions Pomodoro sur la même action sans la marquer comme terminée :

1. Lancez un Pomodoro → 25 min de travail
2. Pause automatique → 5 min (non comptabilisées)
3. Relancez un Pomodoro → 25 min supplémentaires
4. Quand vous cochez l'action comme **DONE** → **50 min** sont enregistrées

<Tip>
  Les pauses entre les Pomodoros ne comptent pas. Seul le temps de travail réel est enregistré sur vos actions.
</Tip>

## Presets disponibles

| Nom | Travail | Pause |
|-----|---------|-------|
| Classique | 25 min | 5 min |
| Long | 50 min | 10 min |
| Court | 15 min | 3 min |
