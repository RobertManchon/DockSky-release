# Roadmap

La Roadmap est le cœur de DockSky. Elle affiche toutes les actions de vos projets, organisées par étapes.

## Naviguer dans la Roadmap

Sélectionnez un projet dans le menu latéral pour afficher sa Roadmap.

## Les actions

Chaque action affiche :
- Son **titre**
- Son **statut** (TODO / DONE)
- Le **temps réel** passé dessus

## Marquer une action comme terminée

Cliquez sur le **cercle à gauche** de l'action. Si un focus ou des sessions Pomodoro étaient en cours sur cette action, le temps est automatiquement enregistré.

## Lancer un focus

Cliquez sur **▶** à droite de l'action pour démarrer un chrono de focus dédié à cette action.

<Warning>
  Un seul focus peut être actif à la fois. Démarrer un focus sur une autre action arrête le précédent sans enregistrement.
</Warning>
