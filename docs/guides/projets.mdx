# Projets

Un projet dans DockSky regroupe un ensemble d'étapes et d'actions qui concourent à un objectif commun.

## Créer un projet

Depuis le tableau de bord, cliquez sur **Nouveau projet** et renseignez les informations de base.

## Structure d'un projet

```
Projet
├── Étape 1
│   ├── Action A
│   └── Action B
├── Étape 2
│   ├── Action C
│   └── Action D
```

## Suivi du temps

DockSky agrège automatiquement le temps passé sur chaque action pour vous donner un **total réel par projet**.

<Info>
  Le temps est calculé uniquement sur les phases de travail — les pauses Pomodoro ne sont pas comptabilisées.
</Info>
