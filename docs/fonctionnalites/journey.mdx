# 🗺️ Parcours

La vue **Parcours** est ton historique de progression. Elle affiche chronologiquement toutes les actions et étapes que tu as terminées, avec la possibilité de filtrer par projet, par période, et de voir les milestones.

:::tip Cerveau TDA
Le TDA vit dans le présent. Dans six mois, tu auras oublié tout ce que tu as construit. La vue Parcours est ton archive de progression — elle te rappelle d'où tu viens et ce que tu as bâti.
:::

---

## Ce qu'on y voit

La vue Parcours affiche une **timeline** de tes accomplissements :
- Les **actions complétées**, classées par date
- Les **milestones** — étapes majeures marquées comme importantes dans tes projets
- Le **contexte de chaque item** (projet d'appartenance, date, durée si enregistrée)

---

## Filtres

### Par projet

Sélectionne un projet dans le menu déroulant pour voir uniquement la progression de ce projet. Utile pour des revues de sprint ou des bilans projet.

### Par type

| Filtre | Contenu |
|--------|---------|
| **Tout** | Toutes les actions et étapes terminées |
| **Milestones seulement** | Uniquement les étapes majeures |

### Par période

| Période | Quand l'utiliser |
|---------|-----------------|
| **Tout le temps** | Vue globale depuis le début |
| **Ce mois** | Bilan mensuel |
| **Cette semaine** | Revue de sprint ou hebdo |
| **Aujourd'hui** | Clôture de journée |

---

## Statistiques

En activant le panneau **Stats**, tu obtiens pour la période sélectionnée :
- Nombre total d'actions complétées
- Répartition par projet
- Tendances (jours les plus productifs)

---

## Milestones

Un milestone est une étape que tu as marquée comme "majeure" dans un projet. Dans la vue Parcours, les milestones apparaissent mis en valeur visuellement — ils jalonnent ta timeline et donnent une structure narrative à ta progression.

---

## Parcours vs Liste des accomplis

| Parcours 🗺️ | Liste des accomplis ✅ |
|------------|----------------------|
| Vue chronologique longue durée | Vue quotidienne / hebdomadaire |
| Milestones et contexte riche | Simple liste de tâches |
| Analyse et stats | Motivation immédiate |
| Mémoire de projet | Fermeture de session |
