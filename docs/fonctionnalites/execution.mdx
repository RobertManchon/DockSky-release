# ⚡ Mode Exécution

Le **Mode Exécution** est la vue la plus simple — et souvent la plus puissante — de DockSky. Elle affiche **une seule tâche à la fois**, avec un gros bouton pour la marquer comme faite.

:::tip Cerveau TDA
Quand tout est urgent et que l'écran montre 47 tâches en même temps, le cerveau TDA se fige. Le Mode Exécution résout ça en réduisant le champ de vision à zéro : une tâche. Rien d'autre. Tu décides, tu fais, tu passes à la suivante.
:::

---

## Principe

Le Mode Exécution pioche dans ta liste de tâches et t'en présente une. Tu as trois options :

| Bouton | Action |
|--------|--------|
| 🟢 **GO** | Tu commences cette tâche. Le timer démarre si activé. |
| ✅ **DONE** | La tâche est terminée. Elle disparaît, la suivante s'affiche. |
| ⏭️ **Skip** | Tu passes à la suivante sans la marquer comme faite |

---

## Commencer une session

### 1. Ouvrir la vue Exécution

**Menu Vue** → _⚡ Exécution_

### 2. Choisir son mode

La vue te propose plusieurs façons de séquencer les tâches :

- **Tâches du jour** — les actions planifiées pour aujourd'hui
- **Quick Wins** — les actions courtes faciles à enchaîner
- **Par projet** — toutes les tâches d'un projet spécifique

### 3. Lancer

Clique sur **GO** pour démarrer. La tâche s'affiche en grand. Travaille dessus. Quand c'est fait, clique **DONE**.

---

## Le gros bouton DONE

Le bouton **DONE** est volontairement surdimensionné. C'est le point de satisfaction visuelle du Mode Exécution — cliquer dessus déclenche l'attribution des XP et alimente ta [Liste des accomplis](./done-list).

---

## Timer intégré

Optionnellement, tu peux activer le **timer** dans le Mode Exécution :
- Il mesure le temps passé sur chaque tâche
- Les données alimentent les statistiques de productivité
- Des presets rapides (15 min, 25 min, 45 min) sont disponibles sous forme de pills cliquables

---

## Quand utiliser le Mode Exécution ?

| Situation | Pourquoi ça aide |
|-----------|-----------------|
| Journée qui démarre mal | Enlève le poids de la liste, une chose à la fois |
| Trop de choses à faire | Réduit l'anxiété de décision |
| Session de rattrapage | Enchaîne des Quick Wins facilement |
| Fin de journée | Vide ce qui reste de façon organisée |

---

## Mode Exécution + Pomodoro

Lance le [Pomodoro](./pomodoro) en parallèle pour combiner "quoi faire" (Exécution) et "pendant combien de temps" (Pomodoro). C'est la configuration recommandée pour les sessions de travail difficiles.
