# 📅 Agenda et Rappels

L'**Agenda** et les **Rappels** sont deux outils complémentaires pour gérer le temps et ne rien oublier.

:::tip Cerveau TDA
Le cerveau TDA est mauvais pour se souvenir des choses au bon moment — mais excellent pour les oublier au mauvais moment. L'Agenda et les Rappels externalisent cette fonction : tu délègues à DockSky la responsabilité de te prévenir au bon moment, à toi de travailler.
:::

---

## L'Agenda

L'Agenda affiche une vue mensuelle de tous tes événements. Il te permet de planifier et de naviguer dans le temps.

### Navigation

- Flèches **‹** / **›** pour changer de mois
- Bouton **Aujourd'hui** pour revenir à la date actuelle
- Clic sur un jour pour voir les événements de ce jour dans le panneau latéral

### Créer un événement

Clique sur **＋ Nouveau** dans l'en-tête de l'Agenda. Une fenêtre s'ouvre avec :
- **Titre** de l'événement
- **Date et heure** de début (et optionnellement de fin)
- **Catégorie** pour organiser et filtrer tes événements
- **Description** libre

### Catégories d'événements

Chaque événement appartient à une catégorie que tu choisis. Les catégories sont filtrables depuis la barre en haut de l'Agenda — coche/décoche celles que tu veux afficher.

## Les Rappels

Un rappel est une alerte à une heure précise. Quand l'heure arrive, une notification apparaît dans la vue [Notifications](./notifications).

### Créer un rappel

Le moyen le plus rapide : depuis le **Bandeau permanent**, saisis du texte et choisis une heure. Le rappel est créé en une seconde sans ouvrir l'Agenda.

Depuis la vue Agenda : bouton **＋ Rappel**.

### Gérer ses rappels

La liste des rappels à venir est visible dans le panneau droit de l'Agenda. Tu peux :
- **Modifier** un rappel (clic dessus)
- **Supprimer** un rappel
- **Voir les rappels passés** en faisant défiler l'historique

### Rappels récurrents

Tu peux définir des rappels récurrents (quotidien, hebdomadaire) — utile pour les rituels TDA : revue du soir, check agenda, médication, etc.

---

## Rappels vs Routines

| | Rappels | Routines |
|--|---------|----------|
| **Déclencheur** | À une heure précise | Condition ou récurrence |
| **Usage** | Pense-bête ponctuel | Habitude structurée |
| **Suivi** | Simple notification | Suivi de complétion |
| **Exemple** | "RDV médecin 14h" | "Revue du soir chaque jour" |

## Accéder à l'Agenda

- **Menu Vue** → _📅 Agenda_
- **Bandeau permanent** → bouton calendrier (si disponible dans ton mode actuel)
