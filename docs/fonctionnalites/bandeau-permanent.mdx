# 📌 Bandeau permanent

Le bandeau permanent est la fonctionnalité de **capture rapide** de DockSky. Il reste visible en permanence, **au-dessus de toutes tes autres fenêtres**, quel que soit ce sur quoi tu travailles.

:::tip Cerveau TDA
L'idée disparaît dans les 30 secondes si tu ne la captures pas. Le bandeau est là pour ça — pas besoin de basculer dans une autre app, pas besoin de mémoriser "je noterai ça plus tard". Tu captures maintenant, tu tries ensuite.
:::

---

## À quoi il sert

Le bandeau remplit deux rôles :

1. **Capturer** : saisir une tâche, une idée ou un rappel en quelques secondes, sans interrompre ce que tu fais
2. **Informer** : afficher en permanence les infos clés (heure, notifications en attente, session Pomodoro active, météo de tes projets)

---

## Éléments du bandeau

| Élément | Description |
|---------|-------------|
| **Champ de saisie** | Capture rapide — texte libre, puis choix du type |
| **Bouton 🔔** | Notifications en attente |
| **Indicateur Pomodoro** | Session active, temps restant |
| **Météo projets** | Indicateur visuel de l'état général de tes projets |
| **Heure** | Heure système en temps réel |

---

## Comment capturer

### Capture simple

1. Clique dans le champ de saisie du bandeau (ou utilise le raccourci clavier si configuré)
2. Tape ta tâche ou idée
3. Appuie sur **Entrée** — la tâche est créée comme **action libre**

### Avec contexte (Projet / Action / Étape)

Tu peux préciser le contexte sans quitter le bandeau :

1. Saisis le titre
2. Sélectionne un **Projet** dans le menu déroulant
3. Sélectionne optionnellement une **Action** existante dans ce projet
4. Valide — la tâche est créée et rattachée directement à la bonne étape de ta roadmap

:::note
Si tu ne précises pas de projet, la tâche atterrit dans les [Actions libres](./actions-libres) — tu pourras la placer sur la roadmap plus tard.
:::

### Créer un rappel

Depuis le bandeau, tu peux aussi créer un rappel horodaté :
1. Saisis le texte du rappel
2. Choisis l'heure via le sélecteur de temps
3. Valide — le rappel apparaîtra dans les [Notifications](./notifications) à l'heure dite

---

## Raccourcis clavier

Le raccourci d'activation du bandeau est configurable dans les [Paramètres](./settings). Par défaut, il est accessible via un raccourci global qui fonctionne même quand DockSky est en arrière-plan.

---

## Le bandeau est toujours là

Contrairement aux autres vues de DockSky, le bandeau ne disparaît pas quand tu changes de vue. Il reste ancré en haut de l'écran, même si tu travailles dans un autre logiciel (selon ta configuration de fenêtre flottante).
