# Bienvenue dans DockSky

DockSky est un assistant de productivité conçu pour les cerveaux qui pensent vite, sautent d'un sujet à l'autre, et oublient ce qu'ils avaient en tête dix minutes après.

Mais pas seulement. Ce qu'il résout — externaliser la charge cognitive, garder le fil, ne rien perdre — est utile à tout le monde.

C'est le **curb cut effect** : les aménagements pensés pour les personnes en fauteuil roulant profitent aussi aux poussettes, aux livreurs, aux cyclistes. DockSky a été conçu par un cerveau TDA, pour tous les cerveaux.

---

## Le problème central

> *"Je ne veux plus réexpliquer qui je suis et ce que je fais à chaque nouvelle conversation."*

Chaque fois que tu ouvres une conversation avec ton IA — Claude, ChatGPT, peu importe — tu repars de zéro. Tu réexpliques ton projet, ton contexte, tes décisions passées. C'est épuisant. Pour un cerveau TDA, c'est souvent la raison pour laquelle on abandonne.

DockSky devient ta **mémoire externe persistante** : structurée, consultable, et partageable avec ton IA en un clic.

---

## Ce que DockSky fait

- **Organise** ton travail en Projets → Actions → Étapes
- **Capture** instantanément sans quitter ce que tu fais (bandeau permanent)
- **Mémorise** tes décisions, tes solutions et tes erreurs dans des **Facettes**
- **Brieffe ton IA** avec ton contexte à jour, sans tout réexpliquer
- **Te garde sur les rails** avec un agenda, des routines, un watchdog et un timer focus

---

## Pour qui

| Profil | Ce que DockSky apporte |
|--------|------------------------|
| Cerveaux TDA / atypiques | Externaliser la charge cognitive, interface TDAH-friendly |
| Développeurs | Contexte de projet toujours chargé pour l'IA |
| Créatifs | Mémoire créative persistante (personnages, décisions éditoriales…) |
| Freelances | Vision claire entre plusieurs projets simultanés |
| Professionnels de santé | Outil concret pour accompagner des patients TDA |

## Getting Started

Get started by **creating a new site**.

Or **try Docusaurus immediately** with **[docusaurus.new](https://docusaurus.new)**.

### What you'll need

- [Node.js](https://nodejs.org/en/download/) version 20.0 or above:
  - When installing Node.js, you are recommended to check all checkboxes related to dependencies.

## Generate a new site

Generate a new Docusaurus site using the **classic template**.

The classic template will automatically be added to your project after you run the command:

```bash
npm init docusaurus@latest my-website classic
```

You can type this command into Command Prompt, Powershell, Terminal, or any other integrated terminal of your code editor.

The command also installs all necessary dependencies you need to run Docusaurus.

## Start your site

Run the development server:

```bash
cd my-website
npm run start
```

The `cd` command changes the directory you're working with. In order to work with your newly created Docusaurus site, you'll need to navigate the terminal there.

The `npm run start` command builds your website locally and serves it through a development server, ready for you to view at http://localhost:3000/.

Open `docs/intro.md` (this page) and edit some lines: the site **reloads automatically** and displays your changes.
