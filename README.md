# DockSky — Documentation publique

Ce repo contient la documentation utilisateur de [DockSky](https://docksky.fr), servie via [Mintlify](https://mintlify.com) sur `docs.docksky.fr`.

## Structure

```
docs/
  mint.json          ← Configuration Mintlify (nav, couleurs, logo)
  introduction.mdx   ← Page d'accueil
  quickstart.mdx     ← Démarrage rapide
  changelog.mdx      ← Historique des versions
  guides/
    projets.mdx
    roadmap.mdx
    pomodoro.mdx
    journal.mdx
```

## Contribuer

Les fichiers `.mdx` sont du Markdown standard avec des composants Mintlify optionnels (`<Card>`, `<Steps>`, `<Tip>`, etc.).
