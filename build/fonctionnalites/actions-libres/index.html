🆓 Actions libres

Les Actions libres regroupent toutes les tâches que tu as capturées sans les placer sur un roadmap. Ce sont les actions en attente de planification : elles existent, elles comptent, mais elles ne sont pas encore rattachées à une étape de projet.

:::tip Cerveau TDA Avoir un endroit dédié aux tâches non planifiées, c'est la différence entre "je sais que ça existe quelque part" et "je sais exactement où c'est". Les Actions libres sont ton filet de sécurité. :::

Qu'est-ce qu'une action libre ?​

Une action est dite libre quand elle n'est pas rattachée à une étape de roadmap (step_id vide).

Ça inclut :

  • Les tâches créées rapidement via le bandeau, sans préciser d'étape
  • Les tâches rattachées à un projet mais pas encore planifiées sur la roadmap
  • Les rappels, petits trucs à faire, tâches administratives isolées
  • Tout ce que tu as capturé "pour plus tard"
remarque

Une action libre peut appartenir à un projet — ce n'est pas parce qu'elle n't pas d'étape assignée qu'elle est orpheline. Elle est simplement en attente de planification.

Accéder aux Actions libres​

  • Bandeau permanent → bouton 🆓 (à droite des autres vues)
  • Menu Vue → 🆓 Actions libres
  • Raccourci clavier : disponible via le menu principal

La vue s'ouvre dans la même popup que les autres vues (Roadmap, Agenda, etc.).

Ce que tu peux faire​

Filtrer par statut​

Un menu déroulant permet de n'afficher que les actions d'un statut donné :

OptionAffichage
ToutesToutes les actions libres
À faireStatut TODO
En coursStatut IN_PROGRESS
TerminéesStatut DONE
AbandonnéesStatut ABANDONED

Ajouter une action rapidement​

Le champ en bas de la vue te permet de créer une action libre en un coup :

  1. Tape le titre de l'action
  2. Appuie sur Entrée
  3. L'action est créée immédiatement, sans projet ni étape assignée

Assigner à un projet​

Quand tu sélectionnes une action dans la liste, un panneau latéral apparaît pour l'assigner à un projet :

  1. Clique sur l'action
  2. Choisis un projet dans le menu déroulant
  3. Clique Assigner — l'action rejoint le projet (mais reste sans étape : à toi de la placer sur la roadmap ensuite)
astuce

L'assignation à un projet ne retire pas l'action de la vue — elle reste visible ici tant qu'elle n'a pas d'étape de roadmap. C'est voulu : "a un projet" ≠ "est planifiée".

Actions disponibles par clic droit​

ActionRésultat
✅ Marquer terminéePasse au statut DONE
🚫 AbandonnerPasse au statut ABANDONED
🗑 SupprimerSupprime définitivement l'action

Workflow recommandé​

Capturer via bandeau → Action libre créée
         ↓
  Traiter les actions libres (hebdo ou quand tu as 10 min)
         ↓
  Option A : Assigner à un projet → placer sur roadmap plus tard
  Option B : Traiter directement → marquer terminée
  Option C : Ce n'est plus pertinent → abandonner ou supprimer

Ce flux s'inspire du GTD (Getting Things Done) : la capture est rapide, le traitement est séparé.

Différence avec les Notifications​

Actions libres 🆓Notifications 🔔
Tâches que tu as crééesAlertes générées par le système
Tu les traites quand tu veuxElles signalent quelque chose d'urgent ou en retard
Action intentionnelleInformation passive
